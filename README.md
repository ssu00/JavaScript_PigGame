# JavaScript_PigGame
## PigGame

### 1) 룰 설명
##### Player1과 Player2가 번갈아 주사위를 던진다. 
##### 각 플레이어는 '1'이 나오기 전까지, 주사위를 던지거나 turn을 넘길 수 있다.
##### 플레이어가 던진 주사위의 눈은 플레이어의 점수에 더해지지만, 이때 '1'이 나온다면 이번 turn에 더해졌던 점수는 모두 사라진다. 
##### Player1과 Player2 중 가장 먼저 100점에 도달하는 사람이 이긴다.
